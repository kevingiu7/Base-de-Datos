# Requerimientos del Proyecto Blibioteca
```
 Sistema de Gestión de Biblioteca
La Primaria Justo Sierra desea implementar un sistema para mejorar la forma en que administra su biblioteca. Actualmente, todo se lleva en libretas y hojas sueltas, lo que ha provocado pérdida de datos, libros extraviados y retrasos en las devoluciones.

El objetivo del nuevo sistema es llevar un mejor control de los libros, alumnos, préstamos y devoluciones, todo a través de una base de datos organizada y fácil de usar. A continuación, se describen las operaciones clave que debe considerar el sistema:

📖 Libros
Cada libro debe registrarse con: título, autor, editorial, grado sugerido y cantidad disponible.

Un mismo libro puede tener varias copias disponibles en la biblioteca.

Se debe poder dar de alta, modificar o eliminar libros del inventario.

👦👧 Alumnos
Los alumnos deben registrarse con: nombre completo, grado, grupo y número de lista.

Un alumno puede tener varios préstamos de libros durante el ciclo escolar.

El sistema debe permitir buscar alumnos por nombre o grupo.

👩‍💼 Bibliotecario

Cada bibliotecario estará registrado con: nombre completo, usuario y contraseña.

Un bibliotecario podrá realizar varios préstamos.

El sistema debe permitir distinguir qué bibliotecario atendió cada préstamo o devolución.

Opcionalmente, se puede registrar el turno (mañana/tarde).

📊 Reportes y Estadísticas
El sistema debe poder mostrar:

Qué libros son los más prestados.

Qué alumnos han solicitado más libros.

Libros con pocos ejemplares disponibles (bajo inventario).
```
